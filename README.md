# dfsguard-java

A Java + Spring Boot starter project for demonstrating how to build and secure a tool-using AI agent.

## Included in this starter
- Cursor-friendly markdown docs in `docs/`
- Cursor rules in `.cursor/rules/`
- Minimal Maven + Spring Boot skeleton
- Sample mock data and malicious document examples
- Step-by-step prompts for building the project in Cursor

## How to use
1. Download and unzip this folder.
2. Open the folder in Cursor.
3. Read `AGENTS.md`.
4. Open `docs/06-cursor-prompts.md`.
5. Paste the prompts into Cursor one phase at a time.

## Base package
`com.dfs.dfsguard`
